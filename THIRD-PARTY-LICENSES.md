# Third-Party Licenses

This document lists the licenses of third-party dependencies used by Stone Soup.

## Stone Soup License

Stone Soup is released under the MIT License. See [LICENSE](LICENSE) for the full text.

## License Compatibility Matrix

Stone Soup (MIT licensed) is compatible with the following dependency licenses:

| License Type | Compatible | Notes |
|-------------|------------|-------|
| MIT | Yes | Permissive, same as Stone Soup |
| BSD-2-Clause | Yes | Permissive |
| BSD-3-Clause | Yes | Permissive |
| Apache-2.0 | Yes | Permissive, requires attribution |
| ISC | Yes | Permissive |
| PSF-2.0 | Yes | Python Software Foundation License |
| Zlib | Yes | Permissive |
| CC0-1.0 | Yes | Public domain dedication |
| Unlicense | Yes | Public domain |
| LGPL-2.1+ | Yes | Dynamic linking OK |
| LGPL-3.0+ | Yes | Dynamic linking OK |
| GPL-2.0+ | No | Would require Stone Soup to be GPL |
| GPL-3.0+ | No | Would require Stone Soup to be GPL |

## Python Dependencies

The Python package dependencies and their licenses can be found in the SBOM
(Software Bill of Materials) included with each release.

To generate a current dependency license report:

```bash
pip install pip-licenses
pip-licenses --format=markdown --with-urls
```

## Multi-Language Bindings

### Rust Crate (stonesoup)

Licensed under MIT. Dependencies are listed in `bindings/rust/Cargo.toml`.
License information is available via:

```bash
cargo install cargo-license
cd bindings/rust
cargo license
```

### Node.js Package (@stonesoup/core)

Licensed under MIT. Dependencies are listed in `bindings/nodejs/package.json`.
License information is available via:

```bash
cd bindings/nodejs
npx license-checker --summary
```

### Java Package (uk.gov.dstl:stonesoup)

Licensed under MIT. Dependencies are listed in `bindings/java/pom.xml`.
License information is generated by Maven during build.

### C Library (libstonesoup)

Licensed under MIT. See `libstonesoup/LICENSE`.

## Compliance

Stone Soup follows the [REUSE](https://reuse.software/) specification for
license compliance. To verify compliance:

```bash
pip install reuse
reuse lint
```

The license inventory (SPDX format) is included with each release.
